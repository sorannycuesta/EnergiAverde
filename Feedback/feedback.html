¿Qué logramos con este proyecto?

EnergiAVerde busca generar impacto tanto en el aprendizaje técnico como en la comprensión ciudadana de la transición energética.

🌱

Comprensión energética

Cualquier ciudadano puede visualizar el avance global de las energías renovables y entender cuánto falta para una transición completa.

📊

Datos accesibles

57 años de historia energética global, navegable, filtrable y exportable desde una interfaz web sin necesidad de instalar nada.

🏡

Estimación personal

Cada hogar o comunidad puede estimar qué porcentaje de su electricidad proviene de fuentes renovables, según datos reales del dataset.

👩‍💻

Habilidades en programación

Dominio práctico de HTML5, CSS3, JavaScript ES6+ y librerías como Chart.js y PapaParse para construir aplicaciones web reales.

🇨🇴

Colombia en el mapa

Visibilizar a Colombia dentro del contexto global de transición energética, destacando su potencial solar, eólico e hídrico.

🔬

Pensamiento computacional

Capacidad de descomponer un problema complejo (la crisis energética global) en módulos manejables de código funcional.

🚀 Proyecto listo para GitHub

Esta aplicación está lista para ser subida al repositorio de GitHub y entregada en la plataforma Moodle como entregable final del bootcamp.
